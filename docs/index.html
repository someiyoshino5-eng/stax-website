Stax Asset — Features

長期投資家のために設計された、
シンプルで使いやすい資産管理。

Stax Assetは資産をシンプルに可視化するための、こだわりの機能を搭載しています。

📈

資産推移グラフ

月次・年次・全期間で資産の成長を確認。折れ線グラフで一目瞭然に把握できます。

🥧

ポートフォリオ円グラフ

円グラフで構成比率をひと目で把握。資産配分のバランスを直感的に確認できます。

✍️

かんたん入力

毎月の資産をすばやく記録。シンプルなUIで入力の手間を最小限に抑えます。

🎨

カスタマイズ

テーマカラーや表示方法を自由に変更。あなただけのダッシュボードを作れます。
